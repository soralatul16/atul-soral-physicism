Gravitational Force Simulation

D.1 Gravitation
F = G × m₁ × m₂ / r²  |  G = 6.674 × 10⁻¹¹ N·m²/kg²

Click 🔴 Record to capture data points

What This Shows

Multiple objects attract each other via gravity: F = Gm₁m₂/r². Drag objects to reposition. Add up to 6 objects and watch the net forces update in real time.

Controls

Drag & drop: Click and drag any object on the canvas.
Add objects: Click the "+ Add" pill to add more masses (up to 6).
Size: Each object has an adjustable visual size so you can represent different bodies.
Shape: Choose circle, car, Sun, Earth, or Moon rendering per object.
Scale: For planetary presets, 1 canvas unit maps to a large physical distance so F calculates correctly.
Play: Animate gravitational attraction. Record: Save state to data table. Screenshot: Download canvas as PNG.

Guided Activities

Activity 1: Start with 2 equal masses. Record F, then double one mass and record again. Verify F doubles.
Activity 2: Drag objects to different distances. Record F at r = 2, 4, 6, 8 units. Plot F vs r — verify the inverse-square law.
Activity 3: Select Earth–Sun preset. Note how the Sun is much larger. Compare the forces to everyday scales.
Activity 4: Add a third object and press Play. Watch the three-body interaction — which object accelerates fastest?
Activity 5: Use the Cars preset. Notice the car shapes. The gravitational force between two cars is incredibly tiny!
