The Principle of Superposition

When two or more waves meet at a point, the total displacement at that point is the vector sum of the individual displacements each wave would produce there.

  • Displacements add algebraically — one up, one down partially cancel; both up reinforce.
  • After overlapping, the waves pass through each other unchanged — they emerge as if nothing happened.
  • Superposition applies to all waves: water, sound, light, waves on strings.
Interference is the name for the pattern that results when waves superpose. Where they reinforce we call it constructive; where they cancel, destructive.

Constructive & Destructive Interference

Constructive

Waves arrive in phase (crest on crest). Displacements add → larger amplitude. Bright / loud.

Destructive

Waves arrive in anti-phase (crest on trough). Displacements subtract → smaller (or zero) amplitude. Dark / quiet.

Total constructive vs destructive (equal amplitudes)

Constructive: phase difference $= 0, 2\pi, 4\pi\ldots$ (whole number of cycles). Destructive: phase difference $= \pi, 3\pi, 5\pi\ldots$ (odd number of half-cycles).

Energy is conserved. Destructive interference doesn't destroy energy — it redistributes it. Energy missing from dark/quiet regions reappears in the bright/loud regions of the pattern.

Path Difference & Phase Difference

For two coherent sources, whether interference is constructive or destructive at a point depends on the path difference — the difference in distance the two waves travel to reach it.

Constructive

$$\Delta d = n\lambda$$

Path difference = a whole number of wavelengths ($n = 0, 1, 2\ldots$). Waves arrive in phase.

Destructive

$$\Delta d = \left(n+\tfrac12\right)\lambda$$

Path difference = an odd number of half-wavelengths. Waves arrive in anti-phase.

Linking path difference to phase difference

$$\Delta\phi = \frac{2\pi}{\lambda}\,\Delta d$$
Reading it: a path difference of one full wavelength corresponds to a phase difference of $2\pi$ (back in phase → constructive); half a wavelength gives $\pi$ (anti-phase → destructive).

Coherence — Why We Need It

A stable, observable interference pattern requires the two sources to be coherent:

  • Same frequency (and hence same wavelength).
  • Constant phase difference — the phase relationship between them does not change with time.
  • Ideally similar amplitude, so cancellation in destructive regions is complete (giving maximum contrast).

Why two lamps don't interfere visibly

Two separate bulbs emit light with rapidly, randomly changing phases. The phase difference at any point fluctuates far too fast to see, so the pattern averages out to uniform brightness. That's why interference demos use a single source split in two (e.g. a double slit) or a laser.

Lasers are prized for interference and holography because they are highly coherent and monochromatic — a single, stable frequency with a fixed phase relationship over long distances.

Worked Examples

Example 1 — type of interference

Two coherent sources (λ = 0.20 m) send waves to a point. The path difference is 0.60 m. Constructive or destructive?

$\Delta d/\lambda = 0.60/0.20 = 3$ — a whole number of wavelengths.

Constructive interference (n = 3).

Example 2 — destructive check

Same sources, path difference now 0.50 m. Constructive or destructive?

$\Delta d/\lambda = 0.50/0.20 = 2.5 = (2+\tfrac12)$ — an odd number of half-wavelengths.

Destructive interference.

Example 3 — phase difference from path

Find the phase difference for a path difference of 0.05 m when λ = 0.20 m.

$\Delta\phi = (2\pi/\lambda)\Delta d = (2\pi/0.20)(0.05) = 2\pi(0.25)$

$= \pi/2 = \mathbf{90^\circ}$

Key Takeaways

  • Superposition: overlapping waves add by vector sum of displacements; then pass on unchanged.
  • Constructive: in phase (crest+crest); amplitude adds; bright/loud.
  • Destructive: anti-phase (crest+trough); amplitude subtracts/cancels; dark/quiet.
  • Path difference: $n\lambda$ → constructive; $(n+\tfrac12)\lambda$ → destructive.
  • Phase link: $\Delta\phi = (2\pi/\lambda)\,\Delta d$.
  • Coherence (same frequency, constant phase difference) is required for a stable pattern; energy is conserved, just redistributed.
